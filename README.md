### Step 1 ‚Äî Clone Repository

```

```

### Step 2 ‚Äî Navigate to Project

```

```

### Step 3 ‚Äî Install Dependencies

```

```

### Step 4 ‚Äî Run Development Server

### Step 5 ‚Äî Run Production Server
